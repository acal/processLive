Z function

This article describes the formula syntax and usage of the Z function in Microsoft Excel.

Description

Returns the compressibility factor for a real gas based on reduced temperature and reduced pressure.

Syntax

Z(reduced_temperature, reduced_pressure)

The Z function syntax has the following arguments:

  • Reduced_temperature     Required. The reduced temperature for the gas (actual temperature / critical temperature).

  • Reduced_pressure     Required. The reduced pressure for the gas (actual pressure / critical pressure).




Remarks

  • Based on linear interpolation of the Nelson-Obert Generalized Compressibility Charts. Accuracy of charts is reported to be 1-2% for Z values greater than 0.6, and 4-6% for Z values 0.3-0.6. Strongly polar gases may have more error.

  • For most gases, reduced pressure is defined as actual temprature divided by critical temperature, and reduced pressure is defined as actual pressure divided by critical pressure.

  • For the gases hydrogen, helium, and neon, reduced temperature is defined as Tr = T/(Tc+8), and reduced pressure is defined as Pr = P/(Pc+8).

  • If the reduced temperature and reduced pressure specified are outside the range of the charts, Z will return a #VALUE! error.
