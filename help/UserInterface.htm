Ribbon Menu

This article describes the function of the Process Utilities Ribbon menu in Microsoft Excel.

Description

The Ribbon menu gives the user access to Process Utilities' Function menu, Conditions menu, and SigFigs menu. It can be found under the ADD-INS heading.





SUBMENUS

Functions

Allows user to insert ProcessUtilities functions using Excel's built-in Function Wizard

Conditions

Allows user to specify values for standard pressure, standard temperature, and ambient pressure.

SigFigs

Allows user to specify and apply rules for how values are displayed in the spreadsheet.
