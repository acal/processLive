GASDENSITY function

This article describes the formula syntax and usage of the GASDENSITY function in Microsoft Excel.

Description

Returns the ideal mass density of the specified gas at the specified conditions.

Syntax

GASDENSITY(unit, temperature, temperature_unit, pressure, pressure_unit)

The GASDENSITY function syntax has the following arguments:

  • Unit     Required. Units for result.

  • Formula     Required. The molecular formula of the gas.

  • Temperature     Required. The temperature of the gas.

  • Temperature_unit     Required. Units for Temperature.

  • Pressure     Required. The pressure of the gas.

  • Pressure_unit     Required. Units for Pressure.



Remarks

  • If the input data types or units are incorrect, GASDENSITY returns the #VALUE! error value.

  • Unit names and prefixes are case-sensitive.

  • Temperature and pressure value/unit pairs can be entered in either order.

=GASDENSITY("lb/ft3", "CH4", 80, "degF", 220, "psig")

Returns the ideal density in lb/ft3 of methane at 80F and 220 psig (0.650).

3

=GASDENSITY("g/cm3", "Ar", 420, "K", 10, "bar")

Returns the ideal density in g/cm3 of argon at 420K and 10 bar (0.0114).
