Conditions Menu

This article describes the function of the Conditions menu in Microsoft Excel.

Description

The Conditions menu allows the user to specify setpoints for Standard pressure, Standard temperature, Elevation, and Ambient Pressure. These are values that are used in other ProcessUtilities functions.



'Set standard conditions' sub-menu

This submenu allows the user to specify values and units for Standard pressure and Standard Temperature. These values are used by functions including CONV, to convert between standard volume units (sL, scf, Mcf, etc.) and molar units (mol, kmol, lbmol, etc.).


'Set elevation or ambient pressure' sub-menu

This submenu allows the user to specify values and units Elevation and Ambient Pressure. These values are used by functions including PCONV, to convert between gauge pressure (psig, barg, etc.) and absolute pressure (psi, bar, mmHg, etc.).
